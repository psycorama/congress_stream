congress_stream
===============

This tool is designed to enable easy access to the streams of the
yearly _Chaos Communication Congress_.

Just start it with

```sh
    ./congress_streams.sh
```

and enjoy the show.


Necessary tools:

 - `xmessage` (Fedora: `gxmessage`)
 - `mpv` or `mplayer`
 - `perl` and `JSON::MaybeXS`
